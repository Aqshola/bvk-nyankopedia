## Tech Stack
- ReactJS
- TailwindCSS
- React lazy-load-image-component
- Jest, React Testing Library
- Cypress
## How to
- `` npm install ``
- Start React -> ``npm start``
- Run Jest -> ``npm run test``
- Run Cypress -> run ``npm start`` then run ``npm run cypress``
